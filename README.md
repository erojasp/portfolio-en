# Eduardo Rojas - Bilingual Corporate CV

Responsive bilingual CV website for GitHub Pages, oriented to industrial instrumentation and process control.

## Files
- `index.html` → main structure
- `styles.css` → visual design
- `script.js` → bilingual logic (EN/ES), navigation state and back-to-top button

## How to publish in GitHub Pages
1. Create a new repository, for example: `eduardo-rojas-cv`
2. Upload these files to the repository root
3. Go to **Settings > Pages**
4. In **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / root
5. Save and wait a few minutes
6. Your site will be available at a URL like:
   `https://your-username.github.io/eduardo-rojas-cv/`

## Notes
- The language selector stores the chosen language in browser local storage
- You can later add a PDF download button or a profile image if needed
